Sample Web App

BT Realty is a fully-functional web app we built and deployed to a Digital Ocean cloud server.

Features

  • Users can register, browse listings, search by category, and make inquiries on properties.

  • Admin dashboard allows staff to easily update site with new property listings, add or update realtors, and monitor the status of user inquiries.

  • User inquiries are automatically emailed to the realtor associated with the property. Users are notified if they have inquired on a listing previously.

  • Seller of the Month is selected form admin dashboard and the about page is dynamically updated.
